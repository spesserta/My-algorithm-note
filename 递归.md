# 递归
所谓递归就是函数自己调用自己，当然需要设置调用的终止条件。<br>
## 一、以求n的阶乘为例，n!=n*(n-1)！，一直扩展下去直到n=1时停止，用函数代码实现就是：<br>
```c++
int fun(int n){
   if(n==1) return 1;  //当n=1时候停止
   return n*fun(n-1);  //返回值为函数自己
}
```
